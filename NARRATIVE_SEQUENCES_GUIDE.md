# 📚 Narrative Sequences - Complete Guide

## What Are Narrative Sequences?

Instead of generating random, disconnected posts, Contai now creates **story arcs** that unfold over 3-5 days.

**Why this matters:** Humans connect with stories, not isolated posts. Sequences create a "rabbit hole" effect where people follow along daily and convert by the end.

---

## 🎯 The Two Sequence Types

### **3-Day Rabbit Hole**

**Best for:** Quick campaigns, product launches, controversial topics

```
Day 1: Stir the Pot (Controversial)
Day 2: The Teach (Solution)
Day 3: Social Proof (Results)
```

#### Day 1: Stir the Pot
**Goal:** Spark debate, get comments, create FOMO

**What it does:**
- Challenges common beliefs
- Calls out industry BS
- Creates curiosity (no solution yet)
- Ends with cliffhanger

**Example Hook:**
```
Most "growth hacks" are bullshit.

I lost $50K testing them all.

Here's what actually works (thread):
```

#### Day 2: The Teach
**Goal:** Establish authority, provide real value

**What it does:**
- References Day 1 controversy
- Provides step-by-step solution
- Uses specific examples/numbers
- Includes your product naturally

**Example Hook:**
```
Yesterday I said most growth hacks are bullshit.

You asked for the real ones.

Here's the exact framework I used to grow from 0 to 10K:
```

#### Day 3: Social Proof
**Goal:** Convert believers into users

**What it does:**
- Shows real results
- Before/after comparison
- Addresses objections
- Clear CTA with urgency

**Example Hook:**
```
This isn't theory.

372 people used this framework last month.

Average result: +2,847 followers

Here's what they're saying:
```

### **5-Day Hero Journey**

**Best for:** Major launches, paradigm shifts, building cult following

```
Day 1: The Problem (Pain)
Day 2: The Enemy (Villain)
Day 3: The Discovery (Breakthrough)
Day 4: The Solution (Framework)
Day 5: The Transformation (Results)
```

#### Day 1: The Problem
**Goal:** Make them FEEL the pain deeply

**Tactics:**
- Share painful story or statistic
- "This is happening to you right now"
- Make it visceral, emotional
- No solution yet (create tension)

**Example:**
```
I checked my bank account.

$847 left.

6 months of building. Zero revenue.

The worst part? I knew exactly why I failed.
```

#### Day 2: The Enemy
**Goal:** Unite against common enemy

**Tactics:**
- Name who/what caused the problem
- "They lied to you about X"
- Build righteous anger
- Create us vs. them

**Example:**
```
The "gurus" sold me a lie.

"Build it and they will come."

Bullshit.

They knew I'd fail. That's how they make money.
```

#### Day 3: The Discovery
**Goal:** Create hope + curiosity

**Tactics:**
- "Then I discovered something..."
- The lightbulb moment
- What everyone else gets wrong
- Tease solution (don't give all yet)

**Example:**
```
Then I found an old forum post.

Some dude made $100K with the "wrong" approach.

No audience. No launch. No ads.

Just one simple framework.
```

#### Day 4: The Solution
**Goal:** Make implementation obvious

**Tactics:**
- Step-by-step framework
- Specific, actionable, detailed
- Include your product as tool
- Screenshots, examples, proof

**Example:**
```
The Framework:

1. Find 10 people with the problem
2. Solve it manually for them
3. Document what worked
4. Turn it into a system
5. Sell the system, not the service

I built Contai using this. Here's how:
```

#### Day 5: The Transformation
**Goal:** Convert desire into action

**Tactics:**
- Before/after comparison
- Specific results (numbers, $, time)
- Multiple testimonials
- Clear CTA with urgency

**Example:**
```
6 months later:

$0 → $23K MRR
847 followers → 12K
Working alone → 3 person team

Same framework. Different outcome.

This could be you in 90 days:
[Your URL]
```

## 🚀 How to Use

### **Command:**
```bash
# 3-Day Sequence
node contai.js sequence "your topic" 3 twitter

# 5-Day Sequence
node contai.js sequence "your topic" 5 linkedin

# With platform
node contai.js sequence "crypto scams" 3 twitter
```

### **Output:**
- JSON file with all days' content
- Each day optimized for its role
- Cross-references between days
- Posting strategy included

### **File Saved:**
```
sequence-your-topic-3day-2026-03-25.json
```

---

## 📊 Posting Strategy

### **Timing:**
- Post at **same time daily** (consistency)
- Best times: 9 AM, 12 PM, or 8 PM local time
- Space posts 24 hours apart

### **Engagement:**
- **Day 1:** Engage HEAVILY with all comments (spark debate)
- **Day 2:** Answer questions, provide extra value
- **Day 3:** Pin the post, respond to DMs

### **Cross-Referencing:**
Each day should reference previous:

**Day 2 opener:**
```
Yesterday I said [X]. You had questions.
Here's the full breakdown:
```

**Day 3 opener:**
```
Day 1: I called out the problem
Day 2: I gave you the solution
Today: Proof it works
```

## 🎯 Best Practices

### **1. Choose Polarizing Topics**
```
✅ Good: "Why most advice is wrong"
❌ Bad: "Tips for success"
```

### **2. Build Tension Early**
```
✅ Good: "I almost quit. Then I found this..."
❌ Bad: "Here's a cool thing I learned"
```

### **3. Be Specific with Results**
```
✅ Good: "+2,847 followers in 30 days"
❌ Bad: "Grew my audience significantly"
```

### **4. Don't Sell Until Day 3 (or 5)**
```
✅ Day 1-2: Pure value
✅ Day 3: Solution + CTA
❌ Day 1: Buy my thing
```
