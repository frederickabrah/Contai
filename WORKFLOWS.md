## Freelancer Finding Gigs

**Who:** Designers, developers, writers, consultants  
**Goal:** Find and convert clients  
**Primary Platform:** Twitter  
**Secondary Platform:** LinkedIn

### Daily Workflow (30 minutes)

#### Morning (15 minutes)

**1. Find Opportunities (5 min)**
```bash
# Search for people looking for your services
node contai.js replies "looking for designer" twitter
node contai.js replies "need a developer" twitter
node contai.js replies "hiring freelancer" twitter
```

**Action:** Post 3-5 helpful replies with portfolio link

**2. Share Authority (5 min)**
```bash
# Teach your niche
node contai.js tutorial "how to choose a brand color palette" twitter
```

**Action:** Post a valuable tutorial from generated content

**3. Build Trust (5 min)**
```bash
# Share a personal journey
node contai.js story "the day I quit my 9-5" twitter
```

**Action:** Post an authentic story to build human connection

**3. Engage (5 min)**
- Reply to everyone who commented on your posts
- DM interested people

#### Afternoon (10 minutes)

**4. Share Work (5 min)**
```bash
# Showcase recent project
node contai.js case "client brand redesign" twitter
```

**Action:** Post case study or work-in-progress

**5. Reply Bait (5 min)**
```bash
# Generate engagement
node contai.js bait twitter
```

**Action:** Post reply bait, respond to all comments

#### Evening (5 minutes)

**6. Follow Up**
- Check DMs
- Reply to inquiries
- Send portfolio links

### Weekly Schedule

| Day | Morning | Afternoon | Evening |
|-----|---------|-----------|---------|
| Mon | Find gigs | Post work | Follow up |
| Tue | Find gigs | Tips thread | Engage |
| Wed | Find gigs | Post work | Follow up |
| Thu | Find gigs | Tips thread | Engage |
| Fri | Find gigs | Case study | Follow up |
| Sat | Light engage | Rest | Rest |
| Sun | Batch content | Schedule | Plan |

### Sunday Batch (20 minutes)

```bash
# Generate 2 weeks of content
node contai.js batch twitter

# Review and save best content
# Schedule throughout the week
```

### Expected Results

| Week | Content Posted | Replies Sent | Leads | Clients |
|------|---------------|--------------|-------|---------|
| 1 | 15 | 25 | 5-10 | 0-1 |
| 2 | 15 | 25 | 10-15 | 1-2 |
| 3 | 15 | 25 | 15-20 | 2-3 |
| 4 | 15 | 25 | 20-30 | 3-5 |

**Monthly Goal:** 3-5 clients, $5K-$15K revenue
