# Changelog

## [2.2.0] - 2026-04-19
### Added
- **Multi-Profile Support**: Use `--brand=name` or `--config=path` to switch niches instantly.
- **URL Scraper**: Repurpose any web article into social media content automatically.
- **Story Archetype**: Generate authentic personal narratives with "Writing DNA."
- **Tutorial Archetype**: Generate step-by-step authority building guides.
- **Lazy Mode**: New `last` command to preview content without opening JSON files.

### Changed
- **Level 2 Humanization**: Enforced Human Writing Laws (Rhythm, Messy Details, Show Don't Tell).
- **JSON-Aware Reflection**: AI can now humanize structured data without breaking code syntax.
- **Brand Agnostic Architecture**: Completely removed hardcoded Rugsnitch references.

### Fixed
- **Model Rotation**: Automatic failover for 429 "Too Many Requests" errors.
- **Sequence Logic**: Fixed argument parsing and narrative continuity bugs.
- **Input Handling**: Better detection of files vs raw text in multiply/audit commands.

## [2.0.0] - 2026-03-29
- Initial production release.
- Modular architecture.
- Real-time news fetching.
- CSV export.
